# Scheduler
Schedule is an app that will allow you to input all of your employees, jobs, shifts, and the number of available shifts for each day into a UI.
It will use this information to quickly output a formatted excel spreadsheet with a schedule that you can present to the employees.

## Built With
C# .Net WPF
